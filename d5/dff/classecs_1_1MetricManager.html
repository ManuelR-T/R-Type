Detailed Description

template<typename MetricKey>
class ecs::MetricManager< MetricKey >

A class for managing multiple metrics identified by a key.

The MetricManager class allows for the addition, removal, and retrieval of metrics using a specified key. This enables organized tracking of various metrics throughout the game.

Template Parameters
MetricKeyThe type representing the key for identifying metrics.
