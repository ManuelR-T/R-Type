Detailed Description

template<typename CommandType, typename PacketType>
class ntw::ResponseHandler< CommandType, PacketType >

A template class for handling network responses.

The ResponseHandler class manages command types and their associated handlers, allowing for the registration and processing of responses received over the network.

Template Parameters
CommandTypeThe type representing command identifiers.
PacketTypeThe type representing the structure of incoming packets.
