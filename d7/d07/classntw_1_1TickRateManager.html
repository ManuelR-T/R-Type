Detailed Description

template<typename T>
class ntw::TickRateManager< T >

A template class for managing tick rates.

The TickRateManager class allows for tracking and managing the frequency of updates for various entities identified by a unique ID. It ensures that updates occur at specified intervals, allowing for precise timing control in systems that require it.

Template Parameters
TThe type representing the unique identifier for each tick rate.
