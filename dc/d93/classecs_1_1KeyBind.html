Detailed Description

template<typename Action, typename Key>
class ecs::KeyBind< Action, Key >

A class for managing key bindings for actions.

The KeyBind template class allows for the association of actions (e.g., player commands) with specific keyboard keys. It provides methods to add, remove, and modify key bindings dynamically.

Template Parameters
ActionThe type representing the action (e.g., an enum for game actions).
KeyThe type representing the key (e.g., sf::Keyboard::Key).
