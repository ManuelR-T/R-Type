Detailed Description

template<typename Component>
class ecs::SparseArray< Component >

A sparse array implementation for managing components in an ECS framework.

The sparse_array class provides an efficient way to store and manage components associated with entities in an Entity-Component-System (ECS) architecture. It uses a vector of std::optional to allow for sparse storage, enabling quick access and modification of components.

Template Parameters
ComponentThe type of component to be stored in the sparse array.
