Detailed Description

Perform health checks on entities within the registry.

Updates the parallax effect in the ECS registry.

Stops the missiles in the ECS registry.

Checks the health of mobs and updates the wave manager accordingly.

This function iterates through the entities in the provided registry and performs necessary health checks on entities marked as local.

Parameters
regReference to the registry containing the entities to be checked.

This function iterates through the entities in the registry, checks their health status, and performs necessary actions such as removing dead mobs and updating the wave manager.

Parameters
regReference to the registry containing the entities.
waveManagerReference to the wave manager responsible for handling waves of mobs.

This function iterates through the entities in the provided registry and performs necessary health checks on entities marked as shared client and server sides.

Parameters
regReference to the registry containing the entities to be checked.
datasToSendReference of a list of data that will be sent to all the clients.

This function is responsible for stopping the missiles that are currently active in the ECS registry. It processes the entities and components related to missiles and kill the missiles entities when needed.

Parameters
regReference to the ECS registry containing the entities and components.
datasToSendReference of a list of data that will be sent to all the clients.

This function is responsible for updating the parallax effect within the given registry. It processes all relevant entities and components to achieve the desired parallax scrolling effect.

Parameters
regReference to the ECS registry.
